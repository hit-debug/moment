## 📅 [2026-05-06] 현재 진행 중
### 🎨 테마 시스템 및 라우팅 최적화
- **라우팅 구조 정상화:** `app/(app)/quote/today.tsx`를 `index.tsx`로 변경하여 탭 클릭 시 기본 화면으로 설정.
- **배경 이미지 이슈 해결:** 목업 데이터(`mockQuotes.ts`)에 고화질 Unsplash 이미지 URL 추가 및 `expo-image` 적용.
- **테마 커스텀 연동:** `useThemeStore`를 확장하여 사용자가 설정한 배경 이미지와 폰트가 명언 피드에 즉시 반영되도록 구현.
- **스와이프 최적화:** `key` 속성을 활용하여 명언 전환 시 배경 이미지와 폰트가 정상적으로 갱신되도록 처리.

### 🐛 오류 수정 및 안정화
- **긴급 크래시 수정:** `themeStore.ts`에서 `Platform` import 누락으로 발생하던 "Platform is not defined" 레드 스크린 오류 해결.
- **스타일 경고 제거:** `_layout.tsx`의 `shadow*` deprecated 스타일을 `Platform.select`로 분기하여 웹(`boxShadow`) / iOS·Android(네이티브) 환경별로 최적화.

### ✨ 애니메이션 및 저널 UI 개선
- **스와이프 애니메이션 고도화:** 배경 이미지(`ExpoImage`)를 고정하고 명언 텍스트만 스와이프되도록 `Animated.View` 구조를 분리하여 더 자연스러운 화면 전환 연출.
- **저널 모달 감성 코피팅:** 타이틀을 '나만의 기록'으로, Placeholder를 '지금 떠오르는 생각을 적어보세요...'로 변경.
- **저널 모달 레이아웃 개편:** 큰 모달창 대신 키보드 바로 위에 붙는 형태의 한 줄 텍스트 입력창(Single-line)으로 변경하여 심플한 사용자 경험 제공.
- **에뮬레이터 키보드 대응:** 웹 개발자 도구의 모바일 화면(iPhone)에서도 네이티브 환경처럼 키보드 영역 회피가 동작하도록 `KeyboardAvoidingView`의 `behavior="padding"` 조건을 `Platform.OS === 'web'`에도 추가.

### 🧩 플랫폼 대응 및 PRD 정합성 확보
- **윈도우 안드로이드 에뮬레이션:** 윈도우 환경 크롬 개발자 도구에서 안드로이드 에뮬레이션 시에도 키보드 레이아웃이 깨지지 않도록 `KeyboardAvoidingView`의 `behavior` 설정을 보완.
- **저널 월 선택 기능:** 저널 목록 화면에서 월별 필터링(예: 3월, 4월) 로직을 연동하고, 지난달 목업 데이터를 추가하여 시각적 확인이 가능하도록 구현.
- **PRD(기획서) 기반 Empty State 적용:** PRD R-009 명세에 따라, 작성된 저널이 없을 때 표시되는 빈 화면 디자인을 고도화하고 '명언 보러 가기(홈 이동)' 버튼 추가.

### 🛠️ 버그 픽스 및 디테일 최적화 (피드백 반영)
- **명언 홈 헤더/배경 렌더링 수정:** 명언 홈 상단의 가려졌던 테마 버튼과 로고의 `zIndex`, `top` 속성을 조정하여 항상 보이도록 수정하고, 웹 환경에서 Unsplash 이미지가 로드되지 않던 문제(CORS)를 파라미터 보완을 통해 해결.
- **저널 리스트 버그 수정 및 정리:** 웹에서 반응하지 않던 '월 선택' Alert를 모달 하단 Action Sheet UI로 변경하고, 불필요한 FAB 아이콘 삭제. 수정 기능은 당일 작성분만 가능하도록 안내창을 띄웠다가, 사용성을 위해 아예 버튼이 보이지 않도록 숨김 처리 반영.
- **저널 전용 공유 모달 UI 리팩토링:** 불필요한 모드 버튼('명언만 공유' 등)을 삭제하고, 명언과 내 저널 기록이 위아래로 한 화면에 깔끔하게 통합되도록 설계. 화면 높이를 풀스크린에 가깝게 키우고 홈의 현재 테마(배경, 폰트)가 공유 카드에 동일하게 적용되도록 연동. 공유 진입 시 발생하던 `useLocalSearchParams` import 누락 크래시 긴급 수정 완료.
- **스크롤 및 텍스트 줄바꿈 최적화:** 마이페이지 로그아웃 버튼이 가려지지 않도록 하단 `paddingBottom` 확대 및 명언 텍스트가 단어 단위로 자연스럽게 줄바꿈(`word-break: keep-all`) 되도록 속성 적용.

### 🏷️ 용어 통일 및 시스템 테마 개편
- **Moment Look 용어 적용:** 명언 홈 우측 상단 기능명을 "Moment Look"으로 변경하고, 커스텀 모달 내 "테마"라는 워딩을 전부 제거하여 "배경", "폰트", "Look N" 등 직관적인 용어로 통일.
- **시스템 테마 화면 신규 생성:** 마이페이지 설정에서 기존 "테마 설정"을 "시스템 테마"로 변경하고, 진입 시 [시스템 설정 / 라이트 모드 / 다크 모드] 3가지를 선택할 수 있는 프리미엄 UI 화면(`system-theme.tsx`)을 새로 구현.
- **전역 테마 연동:** `themeStore.ts`에 `useThemeColors` 훅을 생성하여, 마이페이지와 저널 메뉴 등 모든 하위 화면이 라이트/다크 모드 상태에 따라 배경, 텍스트, 카드 색상이 실시간으로 전환되도록 디자인 가이드 연동 완료.

### 🚑 긴급 버그 픽스 및 공유 모달 로직 고도화
- **공유 모달 크래시 수정:** `useThemeStore` import 누락으로 인한 앱 멈춤 현상(명언 홈, 저널 모두 발생) 해결.
- **배경 이미지 렌더링 최적화:** Unsplash 이미지 URL 쿼리스트링 조합 시 `?`가 중복되어 발생하는 오류 로직 수정.
- **공유 모달 호출 분기 처리:** 하나의 모달을 공통으로 쓰되, 명언 홈 진입 시(모드 토글, 편지 작성 기능)와 저널 메뉴 진입 시(나의 기록 통합 카드)의 UI 구성요소를 완벽히 분리 렌더링.
- **저널 공유 카드 비율 조정:** 저널 메뉴에서 호출 시 높아진 모달(`92%`)에 맞추어 미리보기 카드의 `width` 비율을 확대하여 시각적 안정감 확보.

### ✨ 디테일 UX 최적화 (가독성 및 사용성)
- **테마 미리보기 제공:** 시스템 테마 설정 메뉴에 디자인 가이드를 반영한 미니 카드 형태의 컬러 팔레트 썸네일을 추가.
- **라이트 모드 가독성 QA:** 폰트 설정, 시스템 메뉴 등에서 하드코딩된 색상을 `textPrimary`, `textSecondary` 등으로 교체하여 흰색 배경에서도 또렷하게 보이도록 수정.
- **저널 수정 상태 스토어 연동:** `journalStore.ts`를 신규 생성하여, 당일 저널 내용을 수정한 후 저장하면 실시간으로 목록에 반영되도록 개선.
- **자연스러운 텍스트 줄바꿈:** 웹 환경에서 명언과 편지가 단어 중간에 잘리지 않도록 `wordBreak: 'keep-all'` 속성을 적용.
- **공유 모달 사이즈 미세 조정:** 공유 모달 카드(Square/Full)의 폭을 하단 액션 아이콘들(인스타그램~복사)의 간격에 딱 맞추어 정렬하고, 모달 `maxHeight`를 `90%`로 줄여 스크롤이 발생하지 않도록 최적화.
